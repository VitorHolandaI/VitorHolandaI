# Custom Keyboards

A collection of custom-built mechanical keyboards using KMK firmware.

## About

These projects document the design, build, and firmware of custom keyboards using KMK (a Python-based keyboard firmware for microcontrollers).

## Builds

- **keyboard2** - Keyboard running KMK firmware
- **teclado3** - Third keyboard iteration
